# Bank-Loan-Risk-Application-Analysis-Dashboard-Excel
Excel-based Bank Loan Application Analytics Dashboard analysing loan applications, borrower profiles, loan performance, and lending risk indicators to support data-driven financial decision-making.

📌 Project Overview

This project presents a Bank Loan Application Analytics Dashboard built using Microsoft Excel to analyse lending data and understand borrower behaviour, loan performance, and risk distribution.

The dashboard helps financial institutions monitor loan portfolio performance, borrower risk profiles, and loan demand trends.

The project demonstrates financial data analysis, loan portfolio reporting, and Excel dashboard development.

📊 Key Business Metrics

Total Loan Applications

Total Funded Amount

Total Amount Received

Average Interest Rate

Average Debt-to-Income Ratio

These KPIs provide an overview of the loan portfolio and lending performance.

📈 Dashboard Sections

1️⃣ Loan Portfolio Overview

KPI cards displaying total applications and funding metrics

Overview of lending performance

High-level portfolio insights

2️⃣ Good vs Bad Loan Analysis

The dashboard categorises loans into:

Good Loans Issued

Bad Loans Issued

This helps identify loan default risks and portfolio quality.

3️⃣ Loan Application by Term

Analysis of loan terms:

36-month loans

60-month loans

This helps understand borrower repayment preferences.

4️⃣ Monthly Loan Application Trends

Loan applications are analysed across months to identify seasonal lending patterns and demand trends.

5️⃣ Borrower Employment Analysis

Loan applications analysed by employment length, including:

Less than 1 year

2–5 years

5–10 years

10+ years

This helps evaluate borrower financial stability.

6️⃣ Geographic Loan Distribution

A state-level map analysis shows loan application distribution across different regions.

This helps identify high-demand lending regions.

7️⃣ Loan Purpose Analysis

Loan applications analysed by purpose:

Debt Consolidation

Credit Card

Home Improvement

Major Purchase

Car

Medical

Wedding

Educational

This helps financial institutions understand why customers apply for loans.

8️⃣ Interest Rate Analysis

The dashboard analyses interest rate variations across loan categories:

Charged Off loans

Current loans

Fully Paid loans

This helps evaluate risk pricing and loan performance.
